how to read the icon?

Icon is split vertically in 2.

  • left: can react live?
  • right: can upload reaction?

Color meaning:

  • green: yes
  • orange: no, delay hasn't passed yet
  • red: no
  • white: unable to tell

Other types of icons:

  • question mark: yt-react-db doesn't have the permissions of this youtuber
  • loading
  • white X with red background: an error occured, please reload page
